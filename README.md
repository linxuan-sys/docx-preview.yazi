# docx.yazi
A Yazi plugin for instant Word document preview.

It uses pandoc to convert `.docx` files to plain text on the fly,
and displays the content directly in Yazi preview panel with zero delay.

## Features
- Instant preview for Microsoft Word `.docx` files
- Convert docx to plain text via pandoc
- No lag, real-time lightweight preview

## Dependencies
- pandoc (required)

## Install
```toml
[dependencies]
"linxuan-sys/docx.yazi" = "*"
```
